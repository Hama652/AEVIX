# AEVIX Dashboard Foundation

This directory contains the web dashboard foundation for AEVIX.

## Structure

- `api/` shared route and contract definitions used by the dashboard backend
- `backend/` FastAPI application foundation
- `frontend/` React + TypeScript dashboard shell

## Scope

This phase provides application startup, routing, configuration, logging, authentication placeholders, websocket infrastructure, and page layouts only.

No training code, dataset logic, model logic, or inference runtime is implemented here.
